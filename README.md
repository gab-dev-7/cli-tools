# CLI Tools Website

A Jekyll-based website showcasing my command-line tools, deployed at [cli.gawindlin.com](https://cli.gawindlin.com).

## Overview

This site serves as a centralized hub for my command-line tool projects, providing an accessible overview of each tool's capabilities, features, and usage. It features consistent styling aligned with my other portfolio sites and offers detailed documentation for each project.

## Projects Featured

### [SEC-SUITE](https://github.com/gab-dev-7/sec-suite)
A comprehensive, multi-functional security testing toolkit designed for security professionals and educational purposes. Combines powerful password cracking capabilities with essential network security tools.

**Key Features:**
- Advanced password attacks (Markov Chain, Brute Force, Dictionary, Rainbow Tables)
- Multi-threading for performance improvements
- Support for modern hash algorithms (bcrypt, scrypt, argon2)
- Network port scanner and reconnaissance
- Advanced keylogger with stealth operation
- Interactive CLI interface

### [Port-Scanner](https://github.com/gab-dev-7/port-scanner)
A lightweight, command-line TCP port scanner written in C. Efficiently scans a specified range of ports on a target IP address using non-blocking sockets.

**Key Features:**
- Efficient scanning with non-blocking `connect()` and `select()` calls
- Configurable timeout per port
- Clean output reporting only open ports and timeouts
- Robust error handling
- IPv4 support

### [Chat-App](https://github.com/gab-dev-7/chat-app)
A basic, single-connection TCP chat application implemented in C demonstrating fundamental socket programming concepts.

**Key Features:**
- Simple server-client communication over TCP
- Real-time message exchange
- Command-line interface
- Graceful connection handling
